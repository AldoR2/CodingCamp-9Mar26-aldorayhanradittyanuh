# Product Overview

The Productivity Dashboard is a client-side web application that helps users manage time and tasks. It provides:

- Real-time clock with date and time-based greetings
- 25-minute focus timer (Pomodoro technique)
- To-do list with task management (add, edit, complete, delete)
- Quick links panel for frequently visited websites

All data is stored locally in the browser using Local Storage API. No backend or server required.

## Target Users

Users seeking a simple, distraction-free productivity tool that works entirely in the browser without requiring accounts or cloud storage.

## Key Principles

- Simplicity: Minimal interface, essential features only
- Privacy: All data stays local, no external services
- Performance: Instant load times, immediate UI feedback (<100ms)
- Accessibility: Works across modern browsers without installation
